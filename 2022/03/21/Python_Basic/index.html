Python 기초 - 기초 문법 정리

Author: Lee GaYeong Date: March 21, 2022  09:00:00 Category: Python

형변환 하기

→ x : str 문자열, y : int 정수형

이런 방법으로 문자열을 정수형, 실수형, 논리형으로 바꿀 수 있다.

인덱싱

  • 문자열에서 원하는 자리값의 문자를 가져온다. → H부터 자리 0으로 자리 6번째인 K가 출력 된다. (공백도 포함하여 계산)

슬라이싱

  • 문자열에서 범위를 지정하여 값을 가져온다. → Hello Kaggle! → Kaggle! → Ka → Hlo

포매팅 format

  • %s : 문자열
  • %c : 문자 1개
  • %d : 정수
  • %f : 부동소수
  • %o : 8진수
  • %x : 16진수
  • %% : Literal %(문자 % 자체)

  • 정렬

    → ‘ hi’

  • 소수점 표현

    → ‘3.4213’ , ‘ 3.4213’

  • format사용

    → I eat 1 soup, two milk and 3 bananas

set

  • 중복이 없는 데이터들의 집합
  • 순서는 존재 / 대문자 - 소문자 순
🍒 ({1, 2, 3}, set, 3)
  • 문자를 입력하면??
🍒 {'H', 'e', 'l', 'o'}
  • 값 추가 / 삭제
🍒 {'H', 'e', 'i', 'l'}

리스트 컴프리헨션

  • for-loop 반복문을 한 줄로 처리

→ 리턴값 적어주고 위에서 하나씩 for문 써주면 된다.

  • list / tuple / set 비교
🍒 ([1, 2, 3, 1, 2, 3], (1, 2, 3, 1, 2, 3), {1, 2, 3})
