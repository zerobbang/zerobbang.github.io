PostgreSQL (WSL2)

Author: Lee GaYeong Date: April 13, 2022  09:00:00 Category: Setting

📎 참고 블로그

  • WSL 터미널에서 진행

  • Ubuntu 패키지 업데이트 및 업그레이드 진행

  • PostgreSQL 설치

  • DB에 접근 가능하도록 활성화

  • 사용자 계정 설정

pgAdmin 설치

  • pgAdmin → 최신 버전 pgAdmin 4 v6.8

  • 설치 시 관리자로 실행 및 install for all users로 설치 진행

  • 서버 설정하기

    • Servers → Register → Server…

    • General Tab → test

    • Connection → Password 입력, Host name 127.0.0.1

      만약 에러가 난다면 다음과 같이 진행한다.

      service 활성화된 상태에서 다음 코드 입력

      new-password 에 자신의 비밀 번호 입력

      다시 바꾼 비밀번호 입력하고 save

DB 생성 및 확인

  • Database 우클릭 → Create

  • dataengineering 노드 → shemas → public → Tables 우클릭 → Create - Table 선택

    • General 다음과 같이 설정

    • Columns 다음과 같이 설정

psql 에서 쿼리 실행

  • psql 실행하면 다음과 같이 된다.

  • DB 조회

  • DB 연결 후 테이블 조회
